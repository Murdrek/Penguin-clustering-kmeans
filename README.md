# Automated Penguin Population Segmentation / Segmentación Automatizada de Poblaciones de Pingüinos

## 🇺🇸 Project Overview
This repository contains an end-to-end **Unsupervised Machine Learning** project focused on the automated segmentation of penguin populations. Using morphometric measurements, the goal is to discover underlying biological patterns and group the populations without pre-existing labels.

### Key Features & Workflow
* **Exploratory Data Analysis (EDA):** Data processing, feature scaling, and visualization.
* **Model Automation:** Custom functions to dynamically fit **K-Means** across multiple cluster ranges.
* **Hyperparameter Optimization:** Evaluation and selection of the optimal $K$ using **Inertia (Elbow Method)** and **Silhouette Coefficient** analysis.

### Tech Stack
* **Language:** Python
* **Machine Learning:** Scikit-Learn (`KMeans`, `silhouette_score`)
* **Data & Viz:** Pandas, NumPy, Matplotlib, Seaborn

## 🇪🇸 Descripción del Proyecto
Este repositorio contiene un proyecto de **Aprendizaje No Supervisado** enfocado en la segmentación automatizada de poblaciones de pingüinos. Utilizando medidas morfométricas, el objetivo es descubrir patrones biológicos subyacentes y agrupar las poblaciones sin la necesidad de etiquetas preexistentes.

### Características Clave y Flujo de Trabajo
* **Análisis Exploratorio de Datos (EDA):** Procesamiento de datos, escalado de características y visualización.
* **Automatización del Modelo:** Funciones personalizadas para ajustar el algoritmo **K-Means** dinámicamente sobre múltiples rangos de clústeres.
* **Optimización de Hiperparámetros:** Evaluación y selección del número óptimo de clústeres ($K$) mediante el análisis de **Inercia (Método del Codo)** y el **Coeficiente de Silueta**.

### Tecnologías Utilizadas
* **Lenguaje:** Python
* **Machine Learning:** Scikit-Learn (`KMeans`, `silhouette_score`)
* **Datos y Visualización:** Pandas, NumPy, Matplotlib, Seaborn
